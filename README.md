# NiceHashQuickMiner.RichPresence

Discord Rich Presence for NiceHash QuickMiner

NiceHash QuickMiner のマイニング状況を Discord に Rich Presence として通知します。  
折角なので先月リリースされたばかりの .NET 6 + WinUI 3 で実装しています。
